# Digital Sustainability – Carbon Footprint Calculator

A simple Excel tool to calculate the carbon footprint of digital devices (laptops, smartphones, etc.) based on power usage, usage hours, and grid carbon intensity. Created for **Unit 41: Digital Sustainability**.

## What This Calculator Does

- Estimates **use‑phase emissions** (electricity consumption)
- Adds **manufacturing** and **end‑of‑life** emissions
- Outputs total kg CO₂e and metric tonnes

## Files

- `Carbon_Footprint_Calculator.xlsx` – the main calculator
- `Unit41_Report.pdf` – full research report (2000+ words)
- `screenshot.png` – preview of the calculator

## Example Calculation (Laptop)

| Parameter | Value |
|-----------|-------|
| Device | Dell Latitude (65W) |
| Hours per day | 8 |
| Days per year | 240 |
| Grid intensity | 0.4 kg/kWh (Pakistan) |
| Manufacturing | 200 kg |
| End of life | 10 kg |

**Result:** Total footprint = **259.92 kg CO₂e** (0.26 tonnes)

## How to Use

1. Download the Excel file.
2. Enter your device’s power (Watts), usage hours, and grid intensity.
3. The calculator automatically updates the results.

## Why This Matters

Digital technologies contribute ~2‑4% of global emissions – similar to aviation. By calculating our own footprint, we can make greener choices (e.g., extend device life, use renewable energy).
